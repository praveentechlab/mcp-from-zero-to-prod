# 🚀 MCP From Zero to Prod

A hands-on, end-to-end repository to **learn, build, and productionize Model Context Protocol (MCP) servers** — starting from simple examples and progressing to advanced, real-world **capstone projects**.

This repo is designed as a **learning + builder journey** for developers, architects, and AI engineers working with **Claude, MCP, and agentic systems**.

---

## 🎯 Repository Goals

- Learn MCP fundamentals from scratch
- Build progressively complex MCP servers
- Integrate MCP with Claude Desktop
- Apply production-grade patterns
- Design and implement real-world capstone projects

---

## 🧠 What You’ll Learn

- MCP concepts & architecture
- Tool design using FastMCP
- Claude Desktop integrations
- Error handling & observability
- Security & scalability patterns
- Agentic workflows using MCP
- Production-ready MCP deployments

---

## ▶️ Getting Started

```bash
git clone https://github.com/praveenlark1985/mcp-from-zero-to-prod.git
cd mcp-from-zero-to-prod
```

Follow instructions inside each folder to run individual MCP servers.

---

## 🧪 How to Use with Claude Desktop

1. Enable MCP in Claude Desktop
2. Configure MCP server in `claude_desktop_config.json`
3. Restart Claude Desktop
4. Invoke tools in **Visual mode**
